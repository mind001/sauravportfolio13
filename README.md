# portfolio

    Portfolio : Machine Operator portfolio

## System Configuration

Please be sure you are using `nodeJS  required`.

## Get Started

Install the repository locally:

```bash
git@github.com:Saurav/portfolio.git
```

## Client Settings

```bash
cd portfolio
cmd npm install
```

#Custom Configurations
```spacebars
    src/config/index.js
```


### Run Client

```bash
cd clinet
npm start
```
